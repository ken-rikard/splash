## Spacing Scale

| Token | Value | Usage |

## Visual Hierarchy

Primary screen (dashboard) focal points, in priority order:

1. **Water level reading** (number) — most prominent, largest type on each river card
2. **Status indicator dot** — color-coded to the five-level danger scale, positioned adjacent to water level
3. **River name** — secondary identification text below the level

---

## Color

### Surface Roles

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | white (#FFFFFF) | Page backgrounds, main surfaces |
| Secondary (30%) | neutral-100 (#F5F5F5) | Cards, sidebar, nav bar |
| Accent (10%) | neutral-900 / blue-600 | CTA buttons, river status indicator dots/bars, active navigation state |
| Destructive | red-600 (#DC2626) | Destructive actions only (none in Phase 2; defined for Phase 3 readiness) |

### River Danger Scale (Semantic)

| Level | Label | Color | Usage |
|-------|-------|-------|-------|
| 1 | Low | green-500 (#22C55E) | Status indicator dot/bar for safe levels |
| 2 | Moderate | yellow-500 (#EAB308) | Status indicator dot/bar for moderate levels |
| 3 | High | orange-500 (#F97316) | Status indicator dot/bar for high levels |
| 4 | Very High | red-500 (#EF4444) | Status indicator dot/bar for very high levels |
| 5 | Extreme | purple-600 (#9333EA) | Status indicator dot/bar for extreme/flood levels |

### River Status Indicators (System)

| Status | Color | Usage |
|--------|-------|-------|
| ok | green-500 | Data is fresh and current |
| stale | amber-500 | Data is older than threshold |
| error | red-500 | Fetch failed |

**Accent reserved for:** CTA buttons (e.g. "View Details"), river status indicator dots/bars (danger level), active navigation state (selected tab/link)
