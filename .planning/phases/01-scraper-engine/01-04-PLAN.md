  <files>src/import/metadata-merger.ts, src/import/cli.ts, tests/import/metadata-merger.test.ts</files>

  <read_first>

    (created Plan 01): src/core/types.ts — RiverEntry
  </read_first>

  <action>
    Create `src/import/metadata-merger.ts`:

    - Export class `MetadataMerger`:
      - `async merge(sources: Partial<RiverEntry>[][]): Promise<RiverEntry[]>`
      - Groups entries by NVE station ID (resolved via the research findings)
      - For each station ID:
        - `id` = `"nve:${stationId}"`
        - `name` = from whichever source has it
        - `alternateNames` = names from other sources that differ
        - `grade` = from nokken (if available) else ''
        - `description` = from nokken (if available) else ''
        - `guideUrl` = from nokken (if available)
        - `dangerLevels` = from hvorerdetvann limits (if available) else [0,0,0,0,0]
        - `enabled` = true (all discovered rivers enabled by default)
        - `sources` = which sources contributed data for this entry
      - For entries that can't be matched to a station ID: create with `id` = source-prefixed id
      - Log merge stats: total entries from each source, matched count, unmatched count, final count

    Create `src/import/cli.ts`:

    - Import `HvorErDetVannMetadataScraper` from `./hvorerdetvann-metadata.js`
    - Import `NokkenNetMetadataScraper` from `./nokken-metadata.js`
    - Import `MetadataMerger` from `./metadata-merger.js`
    - Import `RiverRegistry` from `../core/river-registry.js`

    - `async function main()`:
      - Create scrapers and merger
      - Log "Starting metadata import..."
      - const hvorResult = await new HvorErDetVannMetadataScraper().scrape()
      - const nokkenResult = await new NokkenNetMetadataScraper().scrape()
      - const merged = await new MetadataMerger().merge([hvorResult, nokkenResult])
      - const registry = new RiverRegistry()
      - await registry.save(merged)
      - Log "Import complete: N entries written to data/rivers.json"
    - Run main().catch(console.error)

    - Make the CLI runnable via `npx tsx src/import/cli.ts`

    Create `tests/import/metadata-merger.test.ts`:

    - Test: merges two sources with overlapping stations
    - Test: deduplicates by station ID
    - Test: prefers richer metadata from nokken over hvorerdetvann
    - Test: handles unmatched entries (when station ID unknown)
    - Test: empty sources produce empty result
    - Test: RiverEntry has all required fields after merge
  </action>
